Research

CERN Beamline for Schools (BL4S) March 2026
Built the Python analysis pipeline and performed invariant-mass reconstruction for the 7-student Konatori-135 team. Reconstructed π0 → γγ from lead-glass calorimeter data across carbon, aluminium, and lead targets (A = 12–207). Implemented event selection, Gaussian mass fitting, and log-log regression; validated on Monte Carlo: π0 peak at 135.0 MeV/c2, α = 0.735 ± 0.038 in simulation, consistent with Glauber predictions. Code at github.com/hello2himel/bl4s-konatori-135.
